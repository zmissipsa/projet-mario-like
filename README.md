# projet-mario-like
Jeu de plateforme 2D type Mario en Python avec Pygame


## Installation

* $ pip install -r requirements.txt
* $ python main.py

## Contrôles

* Left/A: Move left  
* Right/D: Move right  
* Space/W: Jump     


## Dépendances	
* pygame	

## Standalone windows build

* $ pip install py2exe
* $ python compile.py py2exe
